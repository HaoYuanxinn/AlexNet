# AlexNet on FashionMNIST (PyTorch)

用 PyTorch 复现 AlexNet，使用FashionMNIST 数据集。

## 项目结构
- `model.py`：AlexNet 网络结构定义
- `model_train.py`：训练脚本
- `model_test.py`：测试脚本
- `data/`：FashionMNIST 数据集
- `best_model.pth`：模型权重
